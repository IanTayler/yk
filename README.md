# yk - yet another kernel for unix-like systems.

## Instructions

Be sure to:

1. Install clang. You can use gcc but you have to install a cross-compiler and set
the appropriate make variables like CC and CFLAGS.
2. Install mtools
3. Install grub-pc-bin
